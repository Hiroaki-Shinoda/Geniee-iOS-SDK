App IDの取得

このガイドでは、管理画面から SDK 配信用 App ID の取得手順を説明します。

前提条件

アプリに配信する広告枠が登録されていることを前提とした手順となります。

  • Media（媒体）登録済み
  • Zone（広告枠）登録済み

未登録場合、「媒体社様向けマニュアル」に従って、広告枠を登録お願いします。

App ID の取得手順

  1. 管理画面にログイン
  2. Zone一覧画面に遷移
  3. 広告タグウィンドウの起動

管理画面にログイン

Zone一覧画面に遷移

Inventoryタブから、Media一覧の「サイト名•アプリ名」リンクをクリックします。

広告タグウィンドウの起動

Zone一覧の「広告タグについて」ボタンをクリックします。

ポップアップしたウィンドウのYour App ID は SDK 配信用 App ID になります。
